# -RK3588-AI-
针对驿站仓储管理人工成本高、效率低、安全管控弱的行业痛点，本作品基于RK3588主控的边缘AI平台，打造智能仓储驿站管理助手。系统融合机器视觉识别、边缘计算，搭载巡逻模块，精准完成快递码识别、仓储空位检测与货物信息智能录入；配套可视化交互界面，实现数据查询、状态监控一体化操作。同时集成消防隐患检测功能，实时监测环境安全参数，险情触发即刻告警，全方位保障仓储运营安全，有效降低人力成本、提升管理效率。

PYQT27的介绍：本程序开头大量使用 import 导入各类库文件，是整个仓储巡检系统运行的基础，所有功能都依赖这些模块实现。
首先导入内置基础库：sys 用于程序全局参数管理、程序退出与进程调度；time 用于线程延时、帧率计算与串口时序控制，保障后台任务稳定运行。
其次导入硬件相关库：serial 是串口通信库，负责上位机与下位机 STM32 通信，接收火灾报警、四路电机状态数据，同时下发巡逻启停、小车运动指令。
核心 UI 部分全部依赖 PyQt5 模块：分别导入窗口、布局、按钮、表格、弹窗等界面控件，同时引入多线程、自定义信号、定时器、键盘事件与图像渲染类。此举可以实现全屏触控界面、无卡顿后台线程、弹窗警告、自定义虚拟键盘等交互功能，避免 UI 卡死。
视觉 AI 模块：cv2 用于读取摄像头视频流、画面预处理；再导入本地自研的 rknn 池管理模块与 YOLOv8 优化推理函数，完成 RKNN 硬件加速目标检测，实现仓储实时视觉巡逻识别。
综上，所有 import 分工明确，组合实现了 PyQt 触控 UI + 串口硬件通信 + OpenCV 视频采集 + RKNN-YOLO 人工智能检测 的完整嵌入式上位机架构。
